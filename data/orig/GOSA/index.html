Downloadable Files Directory

The Report Card and other GOSA data dashboards allow education stakeholders to examine education statistics for individual schools and districts. In some cases, stakeholders may require school- and district-level data for all schools. To meet this need, this page provides downloadable spreadsheets with district- and/or school-level data for all schools across Georgia for the years which are available. Click on the file name in the table below for a full description of business rules for each file.
